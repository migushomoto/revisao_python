```c
while (x > aux) {
  aux = x;
  x++;
  printf("%d", x);
}
```

**a.** 0 ou negativo, dependendo dos modificadores de acesso de x

b. 0

c. Negativo

d. 0 ou negativo, dependendo da arquitetura do processador

e. O loop é infinito e seguirá imprimindo números indefinidamente
